# Key interfaces to use

From src/storage.c (existing):
- `get_default_board_path()` returns `~/.config/kanban-cli/boards/default.md`
- `board_save(Board*, filepath)` creates directory if needed with mkdir 0755
- `board_load(Board*, filepath)` stores filename in board->filename
- `Board` struct has `filename[MAX_PATH]` field

From src/models.h:
- `Board` struct has columns[3], task_count per column, filename, last_modified

  <name>Task 1: Add board list scanning and directory functions</name>
  <files>src/storage.c, src/storage.h</files>
  <action>
Add the following functions to storage.c and declare in storage.h:

1. `get_boards_directory(char* buffer, size_t size)` - Returns `~/.config/kanban-cli/boards/` or from config if available

2. `board_list_boards(char*** boards, int* count)` - Scans boards directory for *.md files, returns array of board names (without .md extension), caller must free

3. `board_create(const char* name)` - Creates new board file at `~/.config/kanban-cli/boards/<name>.md` with 3-column template, returns 0 on success

4. `board_delete(const char* name)` - Deletes board file at `~/.config/kanban-cli/boards/<name>.md`, returns 0 on success

5. `board_exists(const char* name)` - Checks if board file exists, returns 1 if exists

Use existing atomic write pattern from board_save() for board creation.
  </action>
